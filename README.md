# Credit-Debit-Card-comparison-dashboard
Project Overview

This project focuses on analyzing credit and debit card usage across multiple brands using a structured dataset. The objective is to understand customer spending behavior, credit limits, card type distribution, and company-wise performance using Power BI visualizations only — no DAX measures were used.

Tools & Technologies

Power BI Desktop (Data cleaning, modeling, and dashboard creation)

CSV File (Primary dataset)

Power Query Editor (Data transformations)

Key Insights & Features
1. Brand-wise Card Distribution
Count of customers using each card brand
Split between credit and debit card types

2. Credit Limit Analysis
Comparison of credit limits across brands
Identification of brands with higher average limits

3. Customer Spending Patterns
Total spend per customer
Spend comparison across companies

4. Multi-Brand Comparison
Transaction and usage trends
Company-wise performance overview

Dashboard Preview
Steps Performed

1️.Data Cleaning (Power Query)
Removed duplicates
Fixed column formats
Cleaned inconsistent text
Converted credit limit & spend into numeric fields

2️.Data Preparation
Renamed and structured columns
Checked data types
Verified missing values

3️.Dashboard Creation (No DAX Used)
Bar charts for credit limits
Donut charts for card type distribution
Clustered bars for brand comparisons
Slicers for filtering by brand, gender, card type

Key Outcomes

Identified which card brand dominates customer usage
Compared credit limits and spending patterns across companies
Understood demographic and card-type distributions
Built a clean, interactive dashboard without DAX
